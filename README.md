# coingeko
proxy requests to congeko 


## Build the image

```bash 
bash build.sh
```

## Run the image

```bash
bash run.sh
```

## Configure the coins that are requested.

The environmnet variable to configure the coins is `COIN_IDS`

ex:

`COIN_IDS="ripple,stellar,cardano,solana,hedera-hashgraph,ethereum,radium"`
